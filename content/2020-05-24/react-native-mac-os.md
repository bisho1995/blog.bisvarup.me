# Getting started

__scaffolding the project with typescript__

```bash
npx react-native init rn-project --version 0.61.5 --template react-native-template-typescript && 
cd rn2 && 
npx react-native-macos-init && 
cd macos && 
pod install && 
cd .. && 
npx react-native run-macos
```
